  • Added Keychain framework
    This means you can, if you want, deposit your password permanently and securely in your keychain. The Kext Updater asks you for a password again if you should change it. Otherwise you do not need to enter the password for any action.
    If you do not want to use the keychain, you have the option to tell the updater to not use it at the beginning. From then on, for most actions (except for checking and downloading the Kexts, Webdriver, Clover) the password has to be entered again.
